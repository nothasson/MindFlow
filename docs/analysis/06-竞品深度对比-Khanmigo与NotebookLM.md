# MindFlow 竞品调研：Khanmigo 与 Google NotebookLM

> **任务**：深度调研 Khanmigo 和 NotebookLM 竞品，提取个人学习工具功能设计参考和 MindFlow 可优化点
> **执行时间**：2026年4月10日
> **重点**：个人学习工具功能（非社交）

---

## 一、Khanmigo（Khan Academy AI 导师）

### 功能清单

| 功能 | 描述 | 成熟度 |
|------|------|--------|
| 苏格拉底式数学辅导 | 不给答案，反问引导推导 | 5/5 |
| 写作教练 | 引导头脑风暴、草稿反馈 | 4/5 |
| 阅读理解伴侣 | 讨论书籍、提理解力问题 | 4/5 |
| AI 口头考试 | 10 分钟对话面试，追问逻辑 | 4/5 |
| 情景记忆（2026新增） | 记住学生弱点，跨学科类比 | 3/5 |
| 自主反馈系统（2026新增） | 要求学生为逻辑辩护 | 3/5 |
| 语音优先学习（2026新增） | 全语音教学会话 | 3/5 |

### 苏格拉底引导的 3 种模式
1. **反问模式**："你认为第一步应该是什么？"
2. **提示模式**："还记得链式法则吗？"（给线索不给答案）
3. **类比模式**（2026新增）：用已掌握知识做类比引导

### 用户反馈
- 好评：教学法评分 9.8/10、$44/年极具性价比、COPPA/FERPA 合规
- 差评：速度慢（设计使然）、学科有限（仅K-12）、需要 Khan 课程背景、纯文本无图像

### MindFlow 可借鉴
1. 对话式考察模式（AI 追问逻辑，不只选择题）
2. 自主辩护机制（追问"你为什么这么认为？"）
3. 跨学科弱点关联类比
4. 教学节奏耐心设计
5. 安全护栏防作弊

### MindFlow 可超越的方面
1. 资料上传解析（Khanmigo 不支持）
2. 知识图谱（Khanmigo 完全没有）
3. 遗忘曲线复习（Khanmigo 无间隔重复）
4. 记忆系统深度（三层 vs 刚起步的情景记忆）
5. 学科通用性（任何资料 vs 仅K-12课程）

## 二、Google NotebookLM

### 功能清单

| 功能 | 描述 | 成熟度 |
|------|------|--------|
| 资料上传与解析 | PDF/Docs/网页/YouTube/音频，每笔记本50来源 | 5/5 |
| 智能问答+精确引用 | 幻觉率减少87%，引用准确率98% | 5/5 |
| 自动摘要 | 摘要+3-5个建议问题 | 5/5 |
| 学习指南生成 | 要点+复习题 | 4/5 |
| 闪卡/测验生成 | 自动生成闪卡和选择题 | 4/5 |
| 音频播客 | 双主播对话式，支持中文 | 5/5 |
| 交互式音频（2026新增） | 播放中可打断提问 | 4/5 |
| Goals 目标化对话 | 设定对话目标和语气风格 | 4/5 |
| 思维导图/信息图/幻灯片 | 自动生成可视化内容 | 3/5 |

### 支持的资料格式
PDF(200MB)、文本文件、Google Docs/Slides、网页URL、YouTube视频、音频文件

### LearnLM 教育模型
- Google 教育专用 AI，基于 Gemini 微调
- Source Grounding：基于源文件回答，引用准确率 98%
- 教育机构免费

### MindFlow 可借鉴
1. **源文件引用锚定**：回复中嵌入"[资料名:第X页]"引用标记
2. **多格式资料支持**：网页URL、YouTube字幕、纯文本
3. **上传后自动概览**：摘要+建议学习问题
4. **教学风格可选**："深入原理"/"通俗比喻"/"考试导向"
5. **音频学习内容**：知识点转化为音频

### MindFlow 可超越的方面
1. 苏格拉底式教学（NotebookLM 只回答不引导）
2. 错误诊断（NotebookLM 不分析错因）
3. 遗忘曲线调度（生成闪卡但无复习调度）
4. 动态知识图谱（NotebookLM 思维导图是静态的）
5. 学习记忆（NotebookLM 不记住学习状态）
6. AI 主动驱动（NotebookLM 是被动工具）
